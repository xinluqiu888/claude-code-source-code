## 功能概述

该模块提供同步版本的Git仓库检测功能，用于需要立即获取结果的场景。

注意：该函数使用`findGitRoot`会遍历文件系统（无子进程），对于异步场景推荐使用`dirIsInGitRepo()`。

---

## 核心内容详解

### 导出函数

- **`projectIsInGitRepo`**
  - 参数：`cwd`（string）- 当前工作目录
  - 返回值：`boolean`
  - 用途：检查目录是否在Git仓库中
