# copy.tsx — 复制最后响应到剪贴板

## 基本信息

| 属性 | 值 |
|------|-----|
| 文件路径 | `/root/projects/claude-code-source-code/src/commands/copy/copy.tsx` |
| 文件类型 | TypeScript React (.tsx) |
| 主要职责 | 提供将 Claude 最后响应复制到剪贴板的功能 |

## 功能概述

该文件实现了 `/copy` 命令，允许用户将 Claude 的最后一条响应复制到系统剪贴板。支持复制第 N 条最近的响应（通过 `/copy N`），为用户提供便捷的方式来保存或分享对话内容。

## 核心内容详解

### 主要功能

**call 函数**
- 接收可选的数字参数 N
- 从消息历史中提取第 N 条助手响应
- 将内容复制到系统剪贴板
- 显示成功或失败提示

### 支持的用法

```
/copy          # 复制最后一条响应
/copy 1        # 同上
/copy 2        # 复制倒数第二条响应
/copy 5        # 复制倒数第五条响应
```

## 设计要点

1. **剪贴板集成**：使用系统剪贴板 API
2. **索引支持**：支持按索引获取历史响应
3. **错误处理**：处理剪贴板访问失败的情况
4. **即时反馈**：操作后立即显示状态提示

## 与其他文件的关系

- **index.ts**: 命令注册
- **剪贴板工具**: 系统剪贴板访问
